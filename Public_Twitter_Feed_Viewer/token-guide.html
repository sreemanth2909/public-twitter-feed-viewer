How to Find Your X.com Authentication Token

Security Warning: Your authentication token provides complete read access to your account. It is like a temporary password. Only share it with people you trust completely. The person you share it with will be able to see your feed exactly as you do.

Overview

To allow a friend to view your X.com feed, you need to provide them with a "token". This isn't a single piece of text, but a small block of code containing three keys from your browser session. Follow the steps below carefully to find them.

Step 1: Open Developer Tools

Go to x.com and make sure you are logged in. Once the page has loaded, open your browser's Developer Tools. You can usually do this by pressing F12 or by right-clicking on the page and selecting "Inspect".

Step 2: Go to the Network Tab

In the Developer Tools window, find and click on the "Network" tab. This tab shows all the network requests your browser makes.

Step 3: Find the "HomeTimeline" Request

In the "Filter" or "Search" bar within the Network tab, type HomeTimeline. This will help you find the specific API request that fetches your feed. You may need to refresh the page or scroll down on your feed for this request to appear.

Click on the request in the list. Its name will look something like AbCdEfGhIjKlMnOpQrSt-HomeTimeline.

Step 4: Copy the Request Headers

With the "HomeTimeline" request selected, look for a new panel that shows details about it. Click on the "Headers" tab within this panel.

Scroll down until you find the Request Headers section. Here you need to find and copy two values:

  • authorization
  • x-csrf-token

Copy the full value for each one, including the word "Bearer" for the authorization header.

Step 5: Find the `auth_token` Cookie

Now, click on the "Cookies" tab for the same "HomeTimeline" request.

Look through the list of cookies until you find the one named auth_token. Copy its corresponding "Value".

Step 6: Assemble the JSON Token

You now have three pieces of information. You need to assemble them into a JSON object. Copy the template below and paste it into a text editor.

{
  "authorization": "PASTE_YOUR_AUTHORIZATION_HEADER_VALUE_HERE",
  "csrfToken": "PASTE_YOUR_X-CSRF-TOKEN_HEADER_VALUE_HERE",
  "authToken": "PASTE_YOUR_AUTH_TOKEN_COOKIE_VALUE_HERE"
}

Replace the placeholder text with the values you copied in the previous steps. Make sure to keep the quotation marks!

Step 7: Share the Token

Your final JSON object is your complete authentication token. Copy the entire block of text and send it securely to the person you want to share your feed with. They will be able to paste this into the X Feed Viewer extension.
